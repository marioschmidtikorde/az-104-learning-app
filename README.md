===================

AZ-104 Learning App

===================

Intent:

Das ist eine App, um für den AZ-104 Kurs zu lernen. Die Fragen aus den Lernpfaden, die man kostenlos bei MS Learn im Internet machen kann, sollen in der questions.yaml abgebildet werden. Die erweiterten Lerngebiete wie "Operational Awareness" und "Scalability" sollen auch inkludiert werden, weil sie im Lernpfad von MS auch behandelt wurden und mit hoher Wsk auch mit abgefragt werden.

Architektur:

Die AZ-104 Container Learn-App ("con-az-104-learning-app") ist in der Subscription "Azure Playground (VSE-Partnerlizenz)" untergebracht.
Die Ressourcengruppe, in der die App liegt, heißt: "rg_mario".
Die Container Registry ist "acrmarioaz104".
